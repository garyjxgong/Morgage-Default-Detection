# Morgage-Default-Detection

## Introduction
The total outstanding value of US mortgages loans was $10.5 trillion by the end 2019 and it is still growing. Therefore, mortgage risk is one of the most important types of risk we should be carefully monitering. The default on mortgage was also the direct casue to 2008 financial crisis. This project aim to investigae features that are crucial in assessing default probability of loan applicants and build classification model to successfully detect high risk applications.
